# ai_medical_assistant

This project is a lightweight AI medical assistant that predicts possible diseases based on user provided symptoms. It also estimates severity levels and provides basic triage guidance along with suggested medicines.

## Highlights
* Predicts disease from symptom input
* Classifies severity (mild / moderate / chronic)
* Provides triage recommendation (Green / Yellow / Red)
* Suggests basic medicines based on predicted disease
* Lightweight ML pipeline (no deep learning / GPU required)
* Easy to run and deploy locally or on cloud

## Tech Stack
Python
Scikit-learn (TF-IDF Vectorization)
XGBoost (Classification Model)
Pandas / NumPy
Joblib (Model Persistence)
